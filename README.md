# Expense Tracking System (C)

## 📌 Overview
This is a console-based Expense Tracking System developed in C. The project helps users manage their daily expenses efficiently by storing and retrieving data using file handling.

## 🚀 Features
- Add new expense records
- View all expenses
- add all the expenses
- search expenses
- Delete expense entries
- Persistent data storage using files

## 🛠️ Technologies Used
- C Programming Language
- File Handling

## 📂 How It Works
The program stores expense details such as:
- Amount
- Category
- Description

All data is saved in a file, allowing users to access it even after restarting the program.

## ▶️ How to Run
1. Compile the program:gcc expense.c
2. a.exe
